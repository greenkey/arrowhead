# About Our Team

Welcome to our **static site generator** documentation. This page will help you understand how the plugin works and what it can do for you.

## Our Mission

We believe in *making knowledge accessible* through simple, fast, and beautiful static websites. Our tool helps you transform your Obsidian vault into a professional website with minimal effort.

## Features

Our plugin provides several key features:

- **Easy to use**: Just configure and publish
- **Customizable themes**: Choose from multiple templates
- **Fast performance**: Static HTML loads instantly
- **Secure**: No server-side dependencies

### Quick Start

To get started, follow these steps:

1. Install the plugin from Obsidian community plugins
2. Configure your settings in the plugin options
3. Click "Generate Static Site" to publish

## Technical Details

Our plugin uses:

- **TypeScript** for type-safe code
- **Vitest** for testing
- **Handlebars** for templating
- **Markdown** for content format
